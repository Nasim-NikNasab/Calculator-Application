# Calculator Application

## About the Project

This project is a simple **Calculator application** developed with **Python** using the **Tkinter** library. It provides a graphical user interface (GUI) that allows users to perform basic mathematical operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* User-friendly graphical interface
* Input validation
* Error messages for invalid input and division by zero

## Technologies Used

* Python
* Tkinter

## How to Run

1. Install Python.
2. Download or clone the project.
3. Run the Python file.
4. Enter two numbers and choose a mathematical operation.
5. Click the **=** button to display the result.
